# Robbie Watch List

A sortable, filterable movie tracker built with React + Vite.

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Deploy to Railway

1. Push this repo to GitHub
2. Go to railway.app and click "New Project"
3. Select "Deploy from GitHub repo"
4. Choose this repo
5. Railway will auto-detect Vite and deploy it
6. Done — you'll get a public URL

## Build for Production

```bash
npm run build
```
